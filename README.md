# University E-Commerce Platform
 
A dynamic e-commerce web application built as a university project, featuring a modern frontend and custom SQL database backend.
 
## 📋 About the Project
 
This project is a fully functional e-commerce platform designed to demonstrate web development skills across frontend and backend technologies. The platform allows users to browse products, manage shopping carts, and complete purchases with a clean, user-friendly interface.
 
## 🛠 Technology Stack
 
- **Frontend**: HTML, CSS, JavaScript
- **Backend Database**: Custom SQL Database
- **Database Management**: SQL queries and transactions
- **Version Control**: Git & GitHub
